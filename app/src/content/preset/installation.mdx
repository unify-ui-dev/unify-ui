## Installation
### Requirements 
Ensure that UnoCSS has been already installed in your project. If not, please follow the [Official UnoCSS installation guide](https://unocss.dev/integrations/vite).

<Steps>
<Step>
### Install PresetUI
<BoxCode title={"Shell"} icon={"shell"} shellStyle>
```bash
npm i -D @unifydev/preset-ui
```
</BoxCode>
Note that PresetUI is a `dev dependency`

</Step>
<Step>
### Add the preset
Inside your `uno.config.(js|ts)` file, import PresetUI and add it in preset List.
<BoxCode title={"uno.config.ts"} icon={"unocss"}>
```js
// import the packages
import { presetUI } from "@unifydev/preset-ui"

export default defineConfig({
  // ...config
  presets: [
    //other presets
    presetUI({
        // your configuration here
    })
  ],
});
```
</BoxCode>
</Step>
</Steps>

Nice you can now use PresetUI in your project, test this :
<BlockCode>
```html
<span class="badge-md ui-solid-red rounded">
    badge md
</span>
```
</BlockCode>
